## 주제 : 유튜브 트렌드 전반 분석 및 모델의 긍,부정 모델 구축 방안

기간 : ```2021년 06월```

#### 실행 환경
```Jupyter notebook```
#### 분석 도구 : ```Python```

### 사용 데이터 
```<Kaggle> - 인기있는 Youtube 동영상 통계 (korea, Japan, America)```

### 🌼 주요 업무 🌼
```
- EDA 분석
- 머신러닝
- PPT 작성
```

### 📝 분석배경
```
코로나19의 영향으로 동영상 플랫폼 수요가 급증하여 유튜브 트렌드 시장을 분석해 의미있는 결과 도출과 유튜브 영상의 긍정 부정 예측 모델을 만들어 대중들의 영상 여론을 어느정도 짐작해봅니다.
```
### 🚩 분석목적
```
본 과제는 인기있는 Youtube 동영상 통계를 이용하여 대중의 관심사를 알아보고 트렌드를 분석합니다.
또한 긍, 부정 예측모델을 통해 대중의 여론을 유추 합니다.
```

### 🚀 구현 목표
```
- 트렌드 데이터를 가지고 특정 카테고리의 트렌드 예측
- 긍,부정 예측 모델 선정 및 구현
```

### 🏃 Project 수행 흐름도
```
- 데이터 수집
  <kaggle 홈페이지> - KR_youtube_trending_data.csv

  
- 분석 설계 및 데이터 전처리
  - 기초자료 분석
    • 상관계수
    • 상관계수 산점도
    • 월별 업로드와 조회수
    • 시간별 업로드와 조회수
    • 각 항목별 TOP 10 도출
    • 각 항목별 TOP 10 도출 및 분석
    
- 모델 구축 및 분석 (머신러닝)
  - 한국 Youtube 영상의 긍/부정 모델 구축
  - target 변수 생성 과정
  - 머신러닝 구축
  - 일본 Youtube 영상의 긍/부정 모델 구축
  - 미국 Youtube 영상의 긍/부정 모델 구축

  - 미국 + 한국 유튜브 모델 구축
```

### ⛳️ 분석 결과
```
- EDA 분석 결과
 (1) 각 항목별 TOP 10에서 공통적으로 보이는 채널
     : HYBE LABELS, Big Hit Labels, BLACKPINK, JYP Entertainment, SMTOWN으로 아이돌 기획사 채널이므로 대중들이 관심은 아이돌을 향하고 있다는 것을 알 수 있음
  (2) 한국의 가장 인기 있는 채널: 스포츠 컨텐츠

- 모델 구축 및 분석 결과
  한국 Youtube: 로지스틱 회귀분석이 가장 도출된 값의 정확동가 70% 내의 성능을 보이므로 채택
  한국파일과 합쳐서 분석하기 합당한 외국 Youtube
  : 미국 + 한국 유튜브 모델이 로지스틱 모델로 가장 설명력이 높아서 해당 모델 채택
```

- 미국 시장의 반응까지 궁금하면 미국 + 한국의 긍,부정 모델을 사용해 파악 가능
